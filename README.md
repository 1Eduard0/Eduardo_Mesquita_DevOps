# ShopMax

Projeto de interface web de um marketplace inspirado em grandes plataformas de e-commerce, como Amazon, desenvolvido apenas com **HTML** e **CSS** para fins de teste e estudo.

## Sobre o projeto

O **ShopMax** é uma página inicial de loja virtual criada para simular a estrutura visual de um site de vendas online. O layout foi pensado para apresentar uma experiência moderna, limpa e organizada, contendo:

- barra superior com logo, busca e navegação
- banner principal de destaque
- seção de categorias
- vitrine de produtos em destaque
- rodapé institucional

Este projeto não possui funcionalidades dinâmicas com JavaScript ou backend. O foco é apenas a construção visual da interface.

## Tecnologias utilizadas

- **HTML5**
- **CSS3**
